pub struct WsFederationApi<C = Client> {
    peers: BTreeSet<PeerId, Global>,
    members: Arc<Vec<FederationMember<C>, Global>>,
    module_id: Option<u16>,
}
Expand description

Mint API client that will try to run queries against all members expecting equal results from at least min_eq_results of them. Members that return differing results are returned as a member faults list.
